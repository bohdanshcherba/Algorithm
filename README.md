# Algorithm
### deap first search
```
Pеалізувати програму, що знаходить цикл у заданому орієнтованому графі з допомогою пошуку у глибину.    


Вхідні дані:
 Перший рядок містить числа N - кількість ребер.
 Наступні N рядків містять пари вершин, що з’єднані між собою ребром.
 
Вихідні дані:
 Перелік вершин, що утворюють цикл або False - якщо циклу немає.
 У графі може бути кілька циклів - достатньо вивести один з них.
```
